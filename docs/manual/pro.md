# 关于 Pro 专业版

为了支持 NebulaLab 的持续良性发展与迭代，我们推出了面向高级科研用户、高保密课题组及工业研发团队的 **Pro 专业版**。

以下是普通版（Standard）与 Pro 专业版的功能与服务差异：

---

## 一、 版本功能对比表

| 功能模块 | 普通版（Standard） | Pro 专业版 |
| :--- | :--- | :--- |
| **基础数据准备** | 支持常规 CSV/TXT/Excel，手动样品整理 | 支持百万行级**极速文件流式导入**、无上限样品管理 |
| **高级数据算法** | 内置基础裁剪、倍数缩放等常规预处理 | **高级多峰拟合（Gauss/Lorentz）、重叠峰智能解析、自适应多算法寻峰** |
| **工作流效率** | 本地单样品手动应用处理模板 | **多核心 CPU 并行计算、全自动批量导入并流式清洗数据** |
| **私有化与生态** | 自由安装社区公开的插件与主题 | **支持企业/课题组内部私有插件市场、局域网安全包分发与管理** |
| **云协作与共享** | 单机运行，项目文件需手动发送传阅 | **实验数据私有云同步、课题组内一键安全共享实验状态** |
| **技术支持与定制** | 社区 GitHub Issues 异步回复 | **24 小时专属算法专家在线支持、特定仪器解析算法定制与集成** |

---

## 二、 如何激活 Pro 专业版

激活 Pro 专业版非常简单，购买授权码后即可实现无缝解锁，无需重新安装软件。

1. **获取授权码**：请访问官方定价页面 `https://nebulalab.io/pricing` 购买授权码 *(仿真服务页面，保护隐私)*。
2. **填写激活码**：启动软件，进入 **“设置” → “许可证管理” (License Management)**。
3. **一键激活**：将您获得的激活码粘贴至输入框中，点击 **“激活”** 按钮。系统将自动连接安全授权服务器进行在线验证。
4. **离线自部署激活**：针对保密等级极高、物理断网的高校重点实验室或军工研究所，我们提供**本地局域网授权服务器部署方案**，请直接联系商业客服及技术支持获取专属包与离线密钥文件。

## 三、 隐私安全承诺

> [!CAUTION]
> **本地优先与科研隐私安全**
> 无论是普通版还是 Pro 专业版，NebulaLab 均坚持**“本地数据绝对隔离”**原则。所有实验仪器的原始数据读取、平滑滤波与出图计算均**完全发生在您的本地计算机上**，除非您主动点击云共享，否则绝对不会有任何数据被秘密上传至外部服务器。我们深刻理解实验数据对于科研工作者的隐私性与前沿价值。
